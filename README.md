# Restaurante Delicioso 🍽️

Sitio web elegante y moderno para un restaurante, desarrollado con HTML, CSS y diseño responsivo.

## 📋 Descripción

**Restaurante Delicioso** es un proyecto web dedicado a ofrecer una experiencia gastronómica excepcional. El sitio presenta un diseño intuitivo y atractivo que permite a los clientes conocer más sobre el restaurante, explorar el menú y contactar para reservas.

## 🌟 Características

- **Diseño Responsivo**: Adaptado para dispositivos móviles, tablets y computadoras de escritorio
- **Navegación Intuitiva**: Menú principal con acceso rápido a las secciones principales
- **Página de Inicio**: Bienvenida con sección de galería y descripción del restaurante
- **Menú Digital**: Presentación de platos con imágenes y descripciones
- **Formulario de Contacto**: Sección para consultas y reservas
- **Branding Consistente**: Logo y colores corporativos en todas las páginas

## 📁 Estructura del Proyecto

```
desarrollador-web/
├── index.html           # Página de inicio
├── menu.html            # Página del menú
├── contacto.html        # Página de contacto
├── images/              # Imágenes del restaurante
│   ├── logo.png
│   ├── chaulafan.jpg
│   ├── ceviche.jpg
│   └── caldo_bola.jpg
└── Style/               # Hojas de estilo CSS
    ├── layout.css       # Estilos generales y layout
    ├── home.css         # Estilos de la página de inicio
    ├── menu.css         # Estilos de la página de menú
    └── contacto.css     # Estilos de la página de contacto
```

## 🚀 Secciones del Sitio

### Inicio (`index.html`)

- Encabezado hero con mensaje de bienvenida
- Galería de imágenes destacadas
- Sección "Sobre Nosotros" con descripción del restaurante

### Menú (`menu.html`)

- Catálogo de platos principales
- Tarjetas interactivas con imágenes y descripciones
- Diseño atractivo para visualizar opciones culinarias

### Contacto (`contacto.html`)

- Formulario de contacto
- Información de contacto múltiple
- Invitación a realizar reservas

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica del sitio
- **CSS3**: Estilos y diseño responsivo
- **Imágenes**: Galería visual de platos

## 📱 Cómo Usar

1. Abre el archivo `index.html` en tu navegador web
2. Navega entre las diferentes secciones usando el menú principal:
   - **INICIO**: Página principal con bienvenida
   - **MENÚ**: Explora nuestros platos
   - **CONTACTO**: Comunícate con nosotros

## 💡 Características del Diseño

- Navegación clara y accesible
- Imágenes de alta calidad
- Tipografía legible
- Colores atractivos y armoniosos

## 🎨 Personalización

Para personalizar el sitio:

- Reemplaza las imágenes en la carpeta `images/`
- Modifica los textos en los archivos HTML
- Ajusta los estilos en los archivos CSS según tus preferencias
